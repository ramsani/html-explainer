# Artifact Ready Goal

Use with `/make-the-right-html` and any generated HTML artifact.

```text
/goal the HTML artifact is ready only when it demonstrably answers the user's primary intent, reduces mental work better than Markdown, shows inspected evidence, separates facts/inferences/assumptions/unknowns, presents recommendation and risk early, includes a specific next action, includes a copy-ready continuation prompt, decides save/private/refresh/supersede/discard, passes available deterministic checks, and leaves no generated junk in git status. Do not finish if the interaction is decorative, the next action is generic, the evidence is vague, the artifact only contains quality labels, or the user would need a second explanation to act.
```
