# Pattern Package Complete Goal

Use when adding or changing a pattern.

```text
/goal the pattern package is complete when the pattern has a clear use case, evidence requirements, artifact structure, failure modes, tie-breakers, acceptance criteria, human-interface guidance, memory decision guidance, and at least one scenario or example proving when to use it and when not to use it. Do not finish if the pattern overlaps another pattern without a tie-breaker or encourages decorative HTML.
```
