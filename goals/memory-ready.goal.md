# Memory Ready Goal

Use when saving or rebuilding local artifact memory.

```text
/goal artifact memory is ready when the artifact has a useful title, intent, summary, freshness risk, privacy class, validity class, tags or topics, next action, find-it-fast text, metadata file, annotation sidecar, and appears in the regenerated knowledge base with a re-entry prompt. Do not finish if the artifact should be discarded, private context is treated as shareable, or the saved item cannot be found again quickly.
```
