# Fact Sheet

Use this before writing claims into HTML.

## Claim Types

Label claims as:

- `verified fact`: read from files, command output, user-provided content, or cited source;
- `inference`: likely conclusion from verified facts;
- `assumption`: used to proceed but not proven;
- `unknown`: not inspected or not knowable from available context.

## Required Sheet

```text
Evidence inspected:
Verified facts:
Inferences:
Assumptions:
Unknowns:
Risk-bearing claims:
Minimum validation:
```

## Rule

Never let visual confidence exceed evidence confidence.

If a claim affects safety, release, data, money, security, or user trust, it must be backed by inspected evidence or marked as unknown.
