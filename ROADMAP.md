# Roadmap

This roadmap is intentionally small. The project should grow by improving trust, examples, and behavior verification before adding more features.

## Next

- Add scenario-based behavior tests for pattern selection and artifact quality.
- Add more before/after demos for common workflows.
- Improve the example gallery so users can understand the value before installing.
- Add screenshots or a short GIF to the README.
- Strengthen semantic checks for example artifacts.

## Later

- Add richer artifact templates only after scenario tests exist.
- Add optional installer presets for different Claude Code setups.
- Add community examples and contribution guidance for new patterns.
- Add a release checklist for tagged public versions.

## Current Principle

Do not expand the system until the current behavior is easier to verify.

The priority is not more HTML. The priority is decision-ready HTML that reliably helps users inspect evidence, understand risk, choose the next action, and continue work with less ambiguity.
