Project identity

html-explainer is a Claude Code skill layer that makes Visual Explainer safer and more useful by applying an evidence-first workflow inspired by Thariq's HTML effectiveness approach.

Its working model is: route → intent distillation → pattern file → UX contract → evidence → fact sheet → artifact → quality bar → decision.
